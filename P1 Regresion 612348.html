Predicción del nivel de adicción al teléfono en adolescentes¶

En este proyecto se utilizará la base de datos Teen Phone Addiction and Lifestyle Survey de Kaggle como fuente principal de información.
El objetivo central es predecir el nivel de adicción al teléfono en adolescentes a partir de diversos factores relacionados con sus hábitos de uso, estilo de vida, rendimiento académico y aspectos de salud mental.

La variable de interés Addiction_Level representa el grado de adicción al celular. Para comprender mejor qué determina este resultado, se analizará el impacto de 22 variables que incluyen horas de uso diario, tiempo de sueño, interacciones sociales, desempeño académico, ansiedad, depresión, entre otras.

A través de modelos de regresión lineal y no lineal, se buscará capturar la relación entre dichas variables y generar predicciones que permitan identificar los factores más influyentes en el nivel de adicción. Este enfoque resulta valioso tanto para comprender patrones de comportamiento adolescente como para proponer estrategias de prevención y concientización en torno al uso saludable de la tecnología.

Las descripciones detalladas de cada variable se encuentran en el repositorio de Kaggle que aloja el dataset.

Finalmente, para comenzar el análisis se importó la base de datos, se eliminaron columnas irrelevantes (Name, Location, School_Grade) y se prepararon los datos para su exploración inicial.

  1. Variables cualitativas

    • Se identificaron 96 outliers, los cuales fueron eliminados para evitar que distorsionaran los resultados del modelo.

  3. Valores nulos

    • Antes de la limpieza no se encontraron valores nulos.
    • Aun así, se aplicó una imputación preventiva con la media de cada variable, dejando la base preparada en caso de que se presentaran huecos.
    • El resultado final mostró 0 valores nulos después de la imputación.

Con estas transformaciones, el dataset quedó limpio y preparado para los siguientes análisis, reduciendo riesgos de sesgos por datos categóricos, atípicos o faltantes.

from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np
import statsmodels.api as sm

X_sel = df_imputed[selected]
y = df_imputed["Addiction_Level"]

lin_reg = LinearRegression().fit(X_sel, y)
y_pred_lin = lin_reg.predict(X_sel)
r2_lin = r2_score(y, y_pred_lin)
rmse_lin = np.sqrt(mean_squared_error(y, y_pred_lin))

poly_model = make_pipeline(PolynomialFeatures(2), LinearRegression())
poly_model.fit(X_sel, y)
y_pred_poly = poly_model.predict(X_sel)
r2_poly = r2_score(y, y_pred_poly)
rmse_poly = np.sqrt(mean_squared_error(y, y_pred_poly))

print("R2 Lineal:", r2_lin)
print("RMSE Lineal:", rmse_lin)
print("R2 Polinomial:", r2_poly)
print("RMSE Polinomial:", rmse_poly)

print("\nCoeficientes Lineales:")
for var, coef in zip(selected, lin_reg.coef_):
    print(f"{var}: {coef}")

X_num = X_sel.apply(pd.to_numeric, errors="coerce").astype(float)
mask = X_num.notnull().all(axis=1) & y.notnull()
X_sm = sm.add_constant(X_num.loc[mask])
y_sm = y.loc[mask].astype(float)
ols = sm.OLS(y_sm, X_sm).fit()
pvals = pd.Series(ols.pvalues, index=X_sm.columns).drop("const")

print("\nVariables con asociación significativa (p<0.05):")
print(pvals[pvals < 0.05].index.tolist())

from sklearn.model_selection import train_test_split
from sklearn.pipeline import make_pipeline

# Variables predictoras y dependiente
X = df_imputed[selected]
y = df_imputed["Addiction_Level"]

# División en entrenamiento (70%) y prueba (30%)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Modelo lineal múltiple
lin_reg = LinearRegression().fit(X_train, y_train)
y_pred_train_lin = lin_reg.predict(X_train)
y_pred_test_lin = lin_reg.predict(X_test)

# Modelo polinomial grado 2
poly_model = make_pipeline(StandardScaler(), PolynomialFeatures(2), LinearRegression())
poly_model.fit(X_train, y_train)
y_pred_train_poly = poly_model.predict(X_train)
y_pred_test_poly = poly_model.predict(X_test)

# Métricas lineal
rmse_train_lin = np.sqrt(mean_squared_error(y_train, y_pred_train_lin))
rmse_test_lin = np.sqrt(mean_squared_error(y_test, y_pred_test_lin))
r2_train_lin = r2_score(y_train, y_pred_train_lin)
r2_test_lin = r2_score(y_test, y_pred_test_lin)

# Métricas polinomial
rmse_train_poly = np.sqrt(mean_squared_error(y_train, y_pred_train_poly))
rmse_test_poly = np.sqrt(mean_squared_error(y_test, y_pred_test_poly))
r2_train_poly = r2_score(y_train, y_pred_train_poly)
r2_test_poly = r2_score(y_test, y_pred_test_poly)

print("Modelo Lineal -> Train RMSE:", rmse_train_lin, "R²:", r2_train_lin,
      "| Test RMSE:", rmse_test_lin, "R²:", r2_test_lin)

print("Modelo Polinomial -> Train RMSE:", rmse_train_poly, "R²:", r2_train_poly,

Conclusión general del proyecto¶

El proyecto permitió analizar y predecir el nivel de adicción al teléfono en adolescentes a partir de la base de datos Teen Phone Addiction and Lifestyle Survey (Kaggle). Tras la limpieza de datos, la selección de variables y la construcción de modelos de regresión, se encontró que el modelo lineal múltiple ofreció un desempeño aceptable (R² ≈ 0.70), mientras que el modelo polinomial de grado 2 logró un ajuste sobresaliente (R² ≈ 0.96, RMSE ≈ 0.26), confirmando su capacidad de generalización en datos de prueba. El análisis inferencial mostró que factores como las horas de uso diario, el tiempo en redes sociales y el tiempo en videojuegos incrementan el nivel de adicción, mientras que las horas de sueño actúan como un factor protector. En conclusión, el estudio demuestra que es posible predecir con gran precisión la adicción al teléfono en adolescentes e identificar los hábitos más determinantes, aportando evidencia útil para diseñar estrategias de prevención y concientización sobre el uso responsable de la tecnología.
